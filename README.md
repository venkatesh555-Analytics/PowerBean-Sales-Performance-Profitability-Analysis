# PowerBean-Sales-Performance-Profitability-Analysis
This project focuses on analyzing shipment-level sales data to monitor overall business performance, operational efficiency, and sales productivity.

The analysis includes:

Monthly sales trend and MoM growth calculation using SQL window functions

Salesperson performance ranking based on total revenue and shipment volume

Product-level demand analysis and contribution to total revenue

Geographic performance comparison

Shipment size distribution analysis using box-based segmentation

The solution combines SQL for data transformation and aggregation with Power BI for interactive visualization, enabling performance monitoring and data-driven decision-making.
